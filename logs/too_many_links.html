So Bookmarks! It's a simple bookmark manager that I made so i could keep my bookmarks and pinned tabs organised aross devices. So far, I just spent a lot of time trying to figure out how to create a new <p> element, allow the user to edit it, and have it save. My first idea was to have a bunch of <p>s hidden and a bunch of text inputs hidden and stacked on top of each other. The input would show up when the user added a new bookmark folder, and I would take what they typed and set the <p> equal to that value and hide the input. This fell apart when I couldn't figure out how to position the <p> and <input> on top of each other (which I ironically figured out by accident later on while working on something completely unrelated).

Eventually, after a lot of trial and error (and a fair bit of googling) I stumbled upon two things: the contenteditable attribute and the insertAdjacentHTML() method. These two things are what I'm currently using to allow users to add bookmark folders. When the new folder button is clicked, I use this jacvascript function to create a new folder:

function addFolder(){
        document.getElementById('one').insertAdjacentHTML('afterend', newFolder);
        localStorage.headerHTML = header.innerHTML;
}

The first line gets the one untitled folder theat is on the sidebar on the left by default and, when the new folder button is pressed, inserts a new untitled contenteditable folder underneath it. The folder name the user enters saves whenever they click anywhere on the page.
